# Chapter 4 Summary

In this chapter, you learned how to work efficiently with the <b>elements in a list.</b>

You learned how to work through a list using a <b>for-loop</b>, how Python uses
<b>indentation errors</b>.

You learned to make simple <b>numerical lists</b>, as well as a few operations you
can perform on numerical lists.

You learned how to <b>slice</b> a list to work with a subset of items and how to
<b>copy lists</b> properly using a slice. ( [:] )

You also learned about <b>tuples</b>, which provide a degree of protection to a
set of values that shouldn't change, and how to <b>style your increasingly complex
code</b> to make it easier to read.
